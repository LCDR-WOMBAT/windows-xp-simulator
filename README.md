# Windows XP Simulator

A nostalgic, fully-functional Windows XP desktop simulator built with vanilla HTML, CSS, and JavaScript. Experience the iconic blue desktop, taskbar, start menu, and classic window management of the early 2000s.

## Features

✨ **Authentic XP Design**
- Classic blue gradient desktop background
- Iconic taskbar at the bottom with Start button
- Nostalgic window chrome and button styling
- Traditional notification area with clock

🖥️ **Functional Desktop**
- Draggable windows with title bars
- Minimize, maximize, and close buttons
- Desktop icons for quick access
- Window state management

📋 **Applications**
- **My Computer** - System information display
- **Notepad** - Simple text editor
- **Start Menu** - Application launcher with shutdown option

🎯 **Interactive Elements**
- Fully draggable windows
- Working window controls
- Responsive desktop icons
- Real-time system clock
- Start menu with hover effects

## Browser Compatibility

Works in all modern browsers:
- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)

## How to Use

1. Open `index.html` in your web browser
2. Click the **Start** button to open the start menu
3. Double-click desktop icons or select from the start menu
4. Drag windows around the desktop
5. Use the window controls to minimize, maximize, or close

## Keyboard Shortcuts

- Click anywhere to close the Start menu
- Drag the window title bar to move windows
- Click window buttons to control window state

## Customization

The simulator is built with vanilla web technologies and can be easily customized:

- Modify CSS in the `<style>` section to adjust colors and styling
- Add new applications by creating new window types in the `openWindow()` function
- Extend functionality with additional JavaScript features
