# Windows XP Desktop Simulator

A fully-functional, client-side Windows XP desktop environment simulator built with HTML5, CSS3, and JavaScript.

## Features

### Core Functionality
✅ **Draggable Desktop Icons** - Move icons around the desktop  
✅ **Start Menu** - Classic Windows XP Start menu  
✅ **Taskbar** - Shows open windows with minimize/restore  
✅ **Draggable Windows** - Move by title bar  
✅ **Window Management** - Minimize, restore, close, z-index  
✅ **Digital Clock** - Real-time clock in taskbar  

### Applications
- **Calculator** - Full arithmetic operations
- **Notepad** - Text editor
- **Paint** - Drawing tools (pencil, eraser, line, shapes, color picker)
- **Minesweeper** - Fully playable game
- **Solitaire** - Card game framework
- **My Computer** - Disk drives and network
- **My Documents** - File browser
- **Internet Explorer** - Retro browser
- **Recycle Bin** - Empty state

### User Experience
✅ **Authentic XP Aesthetics** - Luna theme colors, beveled borders  
✅ **System Sounds** - Retro beeps via Web Audio API  
✅ **Interactive Elements** - Hover effects, active states  

## How to Use

1. Clone and open `index.html` in a web browser
2. Double-click icons to open applications
3. Drag windows/icons to move them
4. Click Start button for the Start menu
5. Right-click Minesweeper cells to flag mines

## Technical Stack
- HTML5, CSS3, JavaScript
- jQuery & jQuery UI (draggable)
- Font Awesome 6 icons
- Web Audio API for sounds
- 100% client-side, no server required

## Browsers Supported
- Chrome/Chromium (latest)
- Firefox (latest)
- Safari (latest)
